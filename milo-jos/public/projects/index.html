Hermes

- Resumen -

Herramienta versátil que interactúa directamente con la API de Inoreader para extraer datos de feeds configurados previamente por el usuario. Tras la extracción, esta solución procesa el contenido eliminando el marcado HTML y normalizando el texto para su posterior clasificación según criterios específicos. La herramienta está diseñada para generar mensajes personalizados para ser enviados a través de la API de Telegram, culminando con la creación de un documento en formato PDF. Este documento sirve como un pseudo-periódico personalizado, permitiendo a los usuarios disfrutar de un resumen informativo adaptado a sus intereses y necesidades.

Ares

- Resumen -

Aplicación web autohospedada y sin acceso a internet, diseñada para funcionar en una red interna, especializada en reconocimiento facial avanzado y con una robusta gestión de usuarios para un acceso seguro y segmentado. Esta herramienta integra tecnologías de vanguardia para crear, gestionar y explotar una base de datos de reconocimiento facial, permitiendo comparaciones precisas de imágenes individuales y grupales. Capaz de procesar lotes de fotografías y extraer rostros de videos, genera documentos descargables que destacan la imagen original, la cara buscada y las ocho coincidencias más similares en la base de datos, ordenadas por similitud.

Babel

- Resumen -

Herramienta avanzada de transcripción autónoma, implementada en una red interna sin acceso a Internet, que aprovecha la tecnología de OpenAI, Whisper, para convertir eficazmente el contenido de audio y video en texto preciso. Incorpora una gestión de usuarios para el acceso seguro y segmentado, mejorando la seguridad y privacidad. Esta solución no solo transcribe sino que también identifica y separa archivos sin contenido auditivo, asegurando una gestión eficiente de archivos. Los usuarios reciben documentos listos para usarse como lectura o subtitulado, con opciones de almacenamiento temporal y eliminación personalizada, resaltando el enfoque en la privacidad y el control total de datos.

Onuris

- Resumen -

Herramienta web diseñada para automatizar y mejorar la gestión de informes operativos. Al cargar un informe, nuestra solución identifica automáticamente entidades de interés, las compara con una base de datos existente y genera un informe en formato CSV. Este informe puede ser fácilmente integrado en sistemas de terceros, facilitando la vinculación de entidades sin la necesidad de intervención manual, optimizando así los procesos operativos y de análisis.
